✨

The Gemini Gem

A custom Gemini Gem trained to read raw discovery notes and extract the structured themes a prototype needs — customer pain points, use case context, key personas, and language from their world.

🤖

Claude as builder

Takes the synthesised context and generates a full HTML/React prototype — tailored copy, customer-specific scenarios, phone frame layout, and a discovery notes section. Not a template. A real build.

📱

Phone frame presentation

The prototype runs inside an iPhone frame on the landing page — so the customer sees their use case on mobile, in the workflow context that matters to them. Scroll, tap, read aloud — all live.

⚡

GitHub Pages, instant

One push, one URL. No Netlify account, no build pipeline, no backend. The page is live and shareable the same session. Anyone with the link can view it — no login required.
